BeMobileProject

This project is a React application that uses styled-components to create a responsive table. The table includes columns for Image (user thumbnail), Name, Job Title, Admission Data, and Phone Number, and has a search function that filters the information by job title, name, and phone number. Furthermore, each row in the table has a pure accordion pattern, created without additional frameworks.
